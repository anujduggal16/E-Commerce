# E-Commerce

## Prerequisites
  
### Create database with name shopdb
#### Username- root 
#### Password- pass
```sql
CREATE DATABASE shopdb;
```
### Run the server file(Node.js must be installed on the system)
```sql
Node server.js
```
### Run Website on address
```sql
http://localhost:5000
```
### Add products on address 
```sql
http://localhost:5000/add_products.html
```
